## 机器学习
周志华教授的[机器学习(西瓜书)](https://book.douban.com/subject/26708119/)这本书中对机器学习的概念进行了解释，其中提到 “**机器学习**是在计算机上从数据中产生 **‘模型’** 的算法，即 **‘学习算法’**，有了学习算法，就可以将经验数据提供给它，并基于这些数据产生模型”
